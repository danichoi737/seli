ROS Package for quadruped robot of powered by DYNAMIXEL.
## Clone and build
````
$ cd ~/<your_ws>/src
$ git clone https://github.com/danichoi737/seli

$ cd ~/<your ws>

# ROS 1
# COMMING SOON

# ROS 2
$ colcon build --symlink-install
$ source install/setup.bash
````
